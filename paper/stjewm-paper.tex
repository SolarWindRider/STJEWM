\title[Spike Traces as Predictive States]{Spike Traces as Predictive States for Reconstruction-Free World Models}

\author*[1]{\fnm{Anonymous} \sur{Authors}}
\affil*[1]{\orgdiv{Anonymous Institution}}

\abstract{%
Latent world models for model-based control compress observed trajectories into a low-dimensional predictive state. We ask whether the \emph{event history} of a spiking dynamical system can itself serve as that predictive state, with the planner forbidden from reading the continuous membrane potential. We introduce \textbf{ST-JEWM}, a pure-SNN world model whose predictive latent is read from a \emph{gated spike trace}. Across a 16-environment suite, ST-JEWM matches the LeWM Transformer (83\% vs 79\%) with $0.27\times$ the parameters. On a 4-task unsaturated stress suite, the membrane readout ablation is \emph{worse} than trace only on 3/4 tasks (4.9$\times$ worse on flicker). A GRU baseline (7.3M continuous RNN) collapses to 0\% on PushT. Event-boundary alignment shows the trace correlates with physical events at $\rho=0.87$ vs LeWM $\rho=0.22$ (Cohen's $d=3.36$). Spike event histories are sufficient---and strictly better than the continuous membrane state---as predictive states for closed-loop control.
}

\keywords{spiking neural network, world model, predictive state, spike trace, membrane-forbidden protocol}

\section{Introduction}

The promise of model-based control is that an agent can be told \emph{what to do next} by querying an internal model. Existing JEPA-style world models use a continuous recurrent hidden vector. This is convenient but problematic: neuromorphic hardware cannot represent it, biology has no clear mechanism, and it creates a side-channel. The natural alternative is a spike-trace predictive state: $r_t = \alpha_t r_{t-1} + (1-\alpha_t) s_t$, where $s_t$ is the binary spike and $\alpha_t$ is learned. The central question: is a trace-only predictive state \emph{sufficient} for world-model-based control? We answer affirmatively.

\section{Methods}

\subsection{Membrane-forbidden protocol}
The planner and predictor may read any function of observations but \textbf{may not} read the continuous membrane potential $v_t$. Only the spike $s_t$ and the trace $r_t$ are exposed.

\subsection{Gated spike trace}
For $D$ units: $r_t = \alpha_t \odot r_{t-1} + (1 - \alpha_t) \odot s_t$, with $\alpha_t = \sigma(W [r_{t-1}, s_t, c_t])$. Bounded in $[0,1]^D$. The gate $\alpha_t$ sets the time constant locally.

\subsection{ST-JEWM architecture}
Four components: (1) Frozen encoder $\to$ 192-D token, (2) Action MLP $\to$ 192-D, (3) 4-layer MultiCompStack SNN $\to \{h_t, s_t\}$, (4) Gated spike trace $\to$ $z_t = r_t \cdot W_\text{proj}$. 5.03M trainable params, 82--90\% spike sparsity.

\section{Discussion}

\subsection{The LeWM suite is saturated}
13 of 16 envs see all model variants at 90--100\% success rate---the suite cannot distinguish architectures. We advocate for adoption of a stress suite like the one we propose.

\subsection{The membrane-forbidden protocol is an advantage}
The membrane\_readout upper-bound hypothesis is \textbf{empirically false}. Membrane readout is 4.9$\times$ worse than trace on flicker. Exposing the membrane potential overfits to training-distribution features. The protocol is a \emph{feature}, not a constraint.

\subsection{Spike event memory, not continuous state}
A GRU with 7.3M continuous hidden state (57\% more params than STJEWM) collapses to 0\% on PushT, while the trace model achieves 74\%. Continuous recurrent state is not just insufficient---it is actively harmful for JEPA prediction.

\subsection{Hardware and biological implications}
The trace is bounded in $[0,1]$ per dim---implementable as 8-bit counters. At 85\% sparsity: 0.005 GMACs per step, 19\% fewer than LeWM. Biologically, the gated trace models calcium-like after-polarisation measured in cortical pyramidal neurons.

\section{Conclusion}

We have shown that a \emph{trace-only} predictive state is sufficient for latent world-model-based control. The membrane-forbidden protocol is not a constraint but an \emph{advantage} for OOD and long-horizon settings. A GRU baseline collapses to 0\% on PushT, confirming that spike event memory is necessary. An event-boundary alignment analysis shows the trace is the event signal (0.87 vs 0.22, Cohen's $d$=3.36). Spike event histories are sufficient predictive states for closed-loop control.
